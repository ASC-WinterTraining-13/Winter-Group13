### TIM定时器配置
- `TIM3`:编码器1(B4+B5)
- `TIM4`:编码器2(B6+B7)
- `TIM5`:PWM波生成
- `TIM6`:按键扫描;编码器读取
- `TIM7`:mpu6050处理标志位 置1;调控(PID相关)周期计数器 ++

### 引脚配置
#### [TB6612电机驱动]
- `VM `:降压模块 6V
- `VCC`:降压模块 3.3V
- `GND`:降压模块 GND
- `AO1`:左电机 电源-(白)
- `AO2`:左电机 电源+(红)
- `BO2`:右电机 电源-(白)
- `BO1`:右电机 电源+(红)
- `GND`:降压模块 GND

- `PWMA`:MM32学习板 A1
- `AIN2`:MM32学习板 A0
- `AIN1`:MM32学习板 A2
- `STBY`:降压模块 3.3V
- `BIN1`:MM32学习板 B10
- `BIN2`:MM32学习板 B11
- `PWMB`:MM32学习板 A3
- `GND `:降压模块 GND

#### [电机1(左)]
- `信号反馈(黄)`:MM32学习板 编码器1区域 B4
- `信号反馈(绿)`:MM32学习板 编码器1区域 B5
- `编码电机(黑)(-)`:MM32学习板 编码器1区域 GND
- `编码电机(蓝)(+)`:MM32学习板 编码器1区域 3.3V-5V
- `电源线(红)(+)`:降压模块 5V电压
- `电源线(白)(-)`:降压模块 GND

#### [电机2(右)]
- `信号反馈(黄)`:MM32学习板 编码器1区域 B6
- `信号反馈(绿)`:MM32学习板 编码器1区域 B7
- `编码电机(黑)(-)`:MM32学习板 编码器1区域 GND
- `编码电机(蓝)(+)`:MM32学习板 编码器1区域 3.3V-5V
- `电源线(红)(+)`:降压模块 5V电压
- `电源线(白)(-)`:降压模块 GND


### 硬件选型
- `电机`:JGA25_370直流减速电机带编码器(6V;620转/分钟) * 2
- `姿态传感器`:(GY-521)MPU6050
- `OLED`:0.96寸SPI通信 128*64 7针脚
- `电池`:航模锂电池
- `核心板`:MM32F327X_G8P + 逐飞学习板
- `电机驱动`:TB6612
- `声光模块`:D7蜂鸣器 + LED
- `降压模块`:LM2596可调降压稳压电源模块
