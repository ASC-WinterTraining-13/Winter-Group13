### TIM定时器配置
- `TIM3`: 编码器1(B4+B5)
- `TIM4`: 编码器2(B6+B7)
- `TIM5`: PWM波生成
- `TIM6`: 按键扫描;编码器读取
- `TIM7`: mpu6050处理标志位 置1;调控(PID相关)周期计数器 ++

### 引脚配置


#### [OLED]
- `GND`: MM32学习板 屏幕接口区域 GND
- `VDD`: MM32学习板 屏幕接口区域 3V3
- `D 0`: MM32学习板 屏幕接口区域 A5
- `D 1`: MM32学习板 屏幕接口区域 A7
- `RES`: MM32学习板 屏幕接口区域 A6
- `D C`: MM32学习板 屏幕接口区域 D0
- `C S`: MM32学习板 屏幕接口区域 A4


#### [MPU60500]
- `VCC`: MM32学习板 姿态传感器区域 3V3
- `GND`: MM32学习板 姿态传感器区域 GND
- `SCL`: MM32学习板 姿态传感器区域 B13
- `SDA`: MM32学习板 姿态传感器区域 B15
- `XDA`: 悬空
- `XCL`: 悬空
- `ADO`: 悬空
- `INT`: 悬空


#### [TB6612电机驱动]
- `VM `: 降压模块 6V
- `VCC`: 降压模块 3.3V
- `GND`: 降压模块 GND
- `AO1`: 左电机 电源-(白)
- `AO2`: 左电机 电源+(红)
- `BO2`: 右电机 电源-(白)
- `BO1`: 右电机 电源+(红)
- `GND`: 降压模块 GND
- `===分隔左右===`
- `PWMA`: MM32学习板 A1
- `AIN2`: MM32学习板 有刷电机区域 A0
- `AIN1`: MM32学习板 有刷电机区域 A2
- `STBY`: 降压模块 3.3V
- `BIN1`: MM32学习板 有来区域 B10
- `BIN2`: MM32学习板 有来区域 B11
- `PWMB`: MM32学习板 A3

- `编码电源(黑)(-)`: MM32学习板 编码器2区域 GND
- `编码电源(蓝)(+)`: MM32学习板 编码器2区域 3V3
- `电机电源(红)(+)`: 降压模块 5V

### 硬件选型
- `核心板`: MM32F327X_G8P + 逐飞学习板
- `电池`: 航模锂电池
- `降压模块`: LM2596可调降压稳压电源模块
- `OLED`: 0.96寸SPI通信 128*64 7针脚
- `姿态传感器`: (GY-521) MPU6050
- `电机`: JGA25_370直流减速电机带编码器 (6V;620转/分钟) * 2
- `电机驱动`: TB6612
- `蓝牙模块`: HC-04蓝牙模块 (SPP2.1+BLE5.0双模;主从一体;3.2V~6V)
- `循迹模块`: 4路循迹模块 (红外对射式)
- `声光模块`: D7蜂鸣器 + LED
